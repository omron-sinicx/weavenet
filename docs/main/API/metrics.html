Metrics¶

weavenet.metric.binarize(m)[source]¶

Binarizes each matrix in a batch into the one-to-one format (if N=M). If N>M, N-M vertices will have no partner and vice versa.

Shape:

  • m: \((B, N, M)\)

  • output: \((B, N, M)\)

Parameters

m (Tensor) – a continously-relaxed assignment between sides \(a\) and \(b\), where |a|=N, |b|=M.

Returns

A binarized batched matrices.

  • output: \((B)\)

Parameters

m (Tensor) – a binary assignment between sides \(a\) and \(b\), where |a|=N, |b|=M.

Returns
